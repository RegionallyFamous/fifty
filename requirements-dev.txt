# purpose (so `python3 bin/check.py` works on a fresh OS install with
# nothing but Python). These deps are only needed when running the
# tests, the lint gate, or the type checker.

# Pinned to ranges (not exact versions) so the gate doesn't break the
# moment a patch release ships; CI's `setup-python` cache uses this
# file's hash, so widening a range invalidates the cache exactly once.

ruff>=0.15,<1

# version Playwright + visual.yml install separately so the lockstep is
